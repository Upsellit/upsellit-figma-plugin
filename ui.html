Export Upsellit package

Instructions

Select one top-level creative frame. The plugin exports display.html.js, display.css, and optional assets/* in a ZIP.

Recommended Figma structure

  • Top-level selected frame contains child frames/groups named p1, p2, and optional p3.
  • Optional shared child groups named email or lead capture export usi_js.display_vars.email_html.
  • Optional child group named phone exports usi_js.display_vars.phone_html.
  • Inside each page, name your CTA layer with cta or button.
  • Name disclaimer layers with disclaimer.
  • Name background layers with background or bg.

Text mapping inside each page

  • First text layer in visual order → usi_head1
  • Second text layer in visual order → usi_head2
  • Remaining text layers → usi_text1, usi_text2, etc.

Export modes

  • Non-flatten: exports the detected background layer as a 3x PNG asset.
  • Flatten: exports the whole selected p1 artboard as a 3x PNG asset.
  • All PNG assets are converted to .webp before the ZIP is downloaded.
  • display.html.js is automatically updated to reference the converted .webp asset paths.
Example layer structure: Selected Frame p1 p2 p3 email phone

Off = export background layer only. On = export full artboard. Both modes export PNG first, then convert to WebP.
